Introduction

Base URL: http://localhost 
This documentation aims to provide all the information you need to work with our API.

<aside>As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).

Authenticating requests

This API is not authenticated.

GET api/user

Example request:
curl --request GET \
    --get "http://localhost/api/user" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"

    "http://localhost/api/user"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

Request      

POST api/users

Headers

Content-Type        

Example: application/json

Accept        

Example: application/json

Body Parameters

email   string     

O campo value deve ser um e-mail válido. Example: qkunze@example.com

password   string     

O campo value deve ter no mínimo 6 caracteres. Example: Z5ij-e/dl4m{o,

tipo   string     

Example: pf

Must be one of:
  • pf
  • pj
nivel   string     

Example: admin

Must be one of:
  • admin
  • simples
status   string     

Example: ativo

Must be one of:
  • ativo
  • inativo
cep   string     

O campo value deve ter 8 caracteres. Example: dqamniih

numero   string  optional    

O campo value deve ter no máximo 20 caracteres. Example: fqcoynlazghdtqtqx

complemento   string  optional    

O campo value deve ter no máximo 255 caracteres. Example: bajwbpilpmufinllwloau
